
## Experiment Insights

### Experiment Purpose
Find out the if the DynaQL agent is learning and which parameters are the best

### Experiment Results
DynaQL Agent is learning. His winrate raises above 97% on SideA and above 89% on SideB.

Best Parameters are:
    - Alpha:        5% - 10%
    - Gamma:        95% - 99%
    - Epsilon:      8% - 20%
    - Memory Size:  -
    - MinMemory:    32

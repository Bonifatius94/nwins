
## Experiment Insights

### Experiment Purpose
Find out the if the SarsaLambda agent is learning and which parameters are the best

### Experiment Results
SarsaLambda Agent is learning. His winrate raises above 92% on SideA and above 88% on SideB.

Best Parameters are:
    - Alpha:        2%
    - Gamma:        99,9%
    - Epsilon:      20%
    - Lambda:       80% - 99,9%
